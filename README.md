# node-typescript-rest-api

## Description
A simple REST API built with Node.js and Express using TypeScript, implementing basic CRUD operations for a user resource.

## What's Built
- Basic user management with CRUD operations.

## How to Run
1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Run `npm run start` to start the server.
5. Access the API at `http://localhost:3000/users`.

## API Endpoints
- `GET /users` - Retrieve all users.
- `POST /users` - Create a new user.
- `GET /users/:id` - Retrieve a user by ID.
- `PUT /users/:id` - Update a user by ID.
- `DELETE /users/:id` - Delete a user by ID.

## Environment Variables
No environment variables are required.
